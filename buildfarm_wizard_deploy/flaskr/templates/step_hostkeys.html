SSH Hostkeys

The hostkey identifies a host when connecting to it using SSH.

We need to specify here the hostkeys of the machines to which the agents will connect. Typically this means entering the hostkey of the repo, as the agents will connect to it to upload the generated packages.

The documentation host is also a typical example.

To retrieve the hostkey for the repo run the following command:

ssh-keyscan repo_hostname

The output will be similar to:

repo ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIA+bpD1Hf9LB+VFN6V6xdRsTi05gPHd0l7LhWbDZpAQM
