Buildfarm configuration wizard

This is a simple web-based wizard to help setting the configuration for a ROS Buildfarm. This will be performed by adjusting the content of your fork of the repository ros_buildfarm_config

We assume that you have previously used the Buildfarm wizard, which generated a config file with all the information you provided. We will be reusing most of these variables.

Introduction

The configuration process aims at adjusting the content of the repository ros_buildfarm_config.

Assumption

We assume that:
  • You already have prepared correctly your custom packages for being inserted into the buildfarm (if not, look here).
  • You already have decided on the type of rosdistro to use (if not, look here).
  • You have already launched the BuildFarm Wizard, and know where is the related configuration file.
